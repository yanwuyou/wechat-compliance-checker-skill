# 微信公众平台运营规范（基础篇 · 关键条款）

> 来源：微信公众平台官方文档（https://mp.weixin.qq.com/）
> 用途：作为合规检查的补充判定依据，覆盖推荐运营规范未包含的条款

## 说明

本文件为《微信公众号和服务号推荐运营规范》（`platform-rules.md`）的补充。推荐运营规范是本文档的子集，专注于"推荐场景"下的内容规范；本文件补充了更基础的账号行为规范和发送内容规范中与文章创作直接相关的条款。

## 第一部分：账号行为规范（与内容创作相关部分）

### 3.3 诱导行为

**3.3.1 诱导分享**
通过外链或公众号/服务号消息等方式，强制或诱导用户将消息分享至朋友圈的行为。涉及商家营销活动且伴有奖励，包括但不限于：实物奖品、虚拟奖品（红包、积分、信息）。

包含以下类型：
- **强制用户分享**：分享后才能继续下一步操作（如分享后方可订阅、分享后方可继续看视频/内容）。
- **利诱用户分享**：分享后对用户有奖励（如邀请好友助力、集赞可获得抽奖机会）。
- **胁迫、煽动用户分享**：用夸张言语胁迫、引诱用户分享（如"不转不是中国人""请好心人转发一下""转发后一生平安""转疯了""必转"等）。

> **本号关注点**：文章结尾 CTA 不能用"分享到朋友圈后截图领取"。

**3.3.2 诱导关注**
通过外链、公众号或服务号群发、搜索或二维码等方式，以奖励或其他方式，强制或诱导用户关注公众号/服务号。奖励方式包括实物奖品、虚拟奖品（红包、积分、信息）等。

包含以下类型：
- **强制或暗示用户关注**：用户需要关注才能获得好处或进行下一步操作。
- **利诱用户关注**：关注后对用户有奖励（如邀请好友助力、投票、关注可获得红包/抽奖机会等）。

> **本号关注点**：工具分享 CTA 不能用"关注后领取""扫码关注获取"。

**3.3.3 违规导流**
- 利用导流场景，在昵称、简介、视频/文章内容等地方以获取更多内容及其他奖励等作为噱头，诱导用户进入公众号/服务号，涨粉之后下发违规内容或进行违规交易。
- 图文以封建迷信、违规擦边（如低俗擦边、标题党擦边）、生僻字/绕过字体对抗等标题吸引用户点击，主要目的是诱导跳转到其他图文页、第三方网页、小程序等。
- 文章使用不完全/擦边的标题和封面、无意义内容，插入诱导性小程序卡片，诱骗用户点击骗取广告收益。

### 3.6 滥用原创声明功能

公众平台鼓励用户就自行创作并发布的作品标识原创，但作品本身应受著作权法保护且不得侵犯他人权益、符合平台运营规范。

**3.6.1 文章滥用原创声明 — 如下情形不得声明原创：**
- **整合引用内容**：未经授权不合理引用或大篇幅整合网络信息、公共信息、报刊杂志、他人作品等。
- **公共公开内容**：法律、法规、决议、通知、历法、通用数表、时事新闻等政府机关/媒体/事业单位发布的文件，或寻人启事、众筹救助、录取分数线、天气预报等。
- **营销宣传内容**：广告宣传、商家活动、单位招聘、投票征稿、互选广告等。
- **非独家授权内容**：虽经原作者授权发布，但未获得独家授权。
- **未获得原著权利人独家授权的再创造内容**：翻译、注释、演讲或采访稿整理、文字音频视频互转等。
- **无意义内容**：胡诌文字、大段文字堆砌、乱凑字数等。
- **恶意隐藏文字的内容**：特殊代码、排版、文字刷白、特殊背景色等恶意规避系统检测。
- **蹭原创内容**：文字含糊简单、无实际意义，主要篇幅为搬运、导流。
- **违法违规内容**：色情、低俗、暴力、血腥、谣言、不实信息等。

> **本号关注点**：AI辅助创作的文章可以声明原创（平台正面案例已确认），但不能直接贴AI生成原文不做加工。引用外部内容不超过20%。

## 与本号的关联总结

《推荐运营规范》（`platform-rules.md`）已覆盖内容创作层面的9条红线。

本文件补充的要点：
1. **3.3 诱导行为** — 补强了分享/关注/导流的具体禁止形式，CTA检查时对照
2. **3.6 原创声明** — AI辅助创作可声明原创，但直接贴AI原文不行
3. **4.24 信息资讯标注来源** — 引用新闻类内容必须标出处
4. **4.17 恶意编辑** — 文章不能过于简短空洞（与7.3信息量不足联动）

其余条款或已被推荐规范覆盖，或本号不涉及。
