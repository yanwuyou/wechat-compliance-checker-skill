# 微信公众号和服务号推荐运营规范

> 来源：微信公众平台官方文档（https://mp.weixin.qq.com/）
> 用途：作为合规检查的判定依据
> 说明：以下为原文全文，包括可选推荐说明、总则&价值观、九条运营规范、其他说明

---

## 可选推荐功能说明

1. 在内容编辑过程中，创作者可选择是否允许内容被推荐，该推荐功能所生效的场景为：公众号消息列表-看一看；内容详情页底部推荐；发现页-看一看入口推荐。可被允许推荐的内容形态包括：图文消息、图片消息、视频消息。
2. 创作者允许内容被推荐时，请注意，这并非表示该内容将会被成功推荐。不符合推荐运营规范和公众平台运营规范的内容将不会被推荐。建议在进行创作时，严格遵循相应规范，确保内容质量，以提高推荐的几率。
3. 转载和分组发表的内容将不会被推荐。
4. 内容发表后，创作者可以在发表记录中手动取消内容被推荐的机会。不过，我们需要提醒你，一旦创作者选择不允许平台推荐，将无法针对该篇内容重新选择允许平台推荐。因此在做出关闭决定前，请充分权衡利弊。
5. 为提高微信公众号和服务号的内容质量，营造良好的创作环境，我们对平台内的内容提出以下倡导和要求。

## 一、总则&价值观

### 我们鼓励倡导的内容
1. 以传递知识、分享经验和展示个人观点为主要目的，有价值、有深度的原创内容。
2. 提供给用户好的阅读体验、精美的设计和排版的内容。
3. 使读者在轻松阅读中收获知识或思考的具有启发性、教育性或者娱乐性质的内容。
4. 金融、教育、医疗卫生、司法等专业人士发布的权威真实信息和专业知识。

### 我们不希望看到的内容
1. 违反法律法规、违反社会主义核心价值观的内容。
2. 不实内容：例如通过捏造或扭曲事实的内容以吸引眼球博取流量的内容。
3. 违背公序良俗、违背科学的、不尊重他人的、令人极度不适等不良内容。
4. 仅以营销、广告为目的，缺乏实际价值的宣传和营销推广，甚至虚假宣传的内容。
5. 通过各种手段绕过平台检测、审核的内容。

## 二、运营规范

创作者的行为或内容如涉嫌违反以下运营规范，一经发现，相应的内容将不被平台推荐，或被平台取消推荐，包括但不限于：

### 1、恶意使用

**1.1 危害平台安全、扰乱秩序的行为，如：**
- 提供破坏产品生态和流程的服务，如指导用户绕过防沉迷系统；
- 钻平台漏洞以赚取流量；
- 假冒、仿冒、捏造政党、党政军机关、企事业单位、人民团体和社会组织以及国家（地区）、国际组织的名称、标识；假冒、仿冒、捏造政党、党政军机关、企事业单位、人民团体和社会组织以及国家（地区）、国际组织的名称、新闻网站、报刊社、广播电视机构、通讯社等新闻媒体的名称、标识等，或擅自使用具有新闻属性的名称，发布相关新闻报道；
- 其他破坏平台秩序，影响平台公平性、真实性或造成其他不利于生态健康发展的行为。

**1.2 恶意违规及对抗行为，如：**
- 使用技术手段规避检测；
- 恶意注册使用公众号/服务号，批量运营；
- 添加无关标签以误导用户。

### 2、侵权内容

**2.1 禁止侵犯他人合法权益的行为，例如：**
- 发布、传播推广盗版内容、资源；
- 抄袭、搬运内容；
- 通过在资料、内容中使用相近文字等来混淆大IP/品牌，误导用户认为其为版权方或授权方等正主，以实现不当目的。

### 3、不实信息

平台鼓励真实、原创内容，不倡导不实信息。本规范所称的不实信息，指没有相应事实基础，通过捏造或扭曲事实、隐瞒真相而产生的信息。包括但不限于：
- 偷换概念、断章取义，歪曲事件事实，可能会引发公众恐慌、扰乱社会秩序的标题和内容；
- 在关注的社会热点或公众关心的事件、人物里编造或传播未证实的信息，以吸引点击增加流量或损害特定个人或群体的内容；
- 对正常社会秩序和安全造成威胁的不实信息或传言；
- 可能造成暴力行为的错误信息和无法证实的传言；
- 利用新技术（如虚拟现实、aigc）制作、发布的虚假信息；
- 未经考证的，无依据揣测的历史内容。

### 4、违背公序良俗

公序良俗通常是指公共秩序与善良风俗。公共秩序，是指政治、经济、文化等领域的基本秩序和根本理念，是与国家和社会整体利益相关的基础性原则、价值和秩序。善良风俗，是指基于社会主流道德观念的习俗，也可称为社会公共道德，是社会成员所普遍认可、遵循的道德准则。

违反公序良俗的内容及行为，包括但不限于：

**4.1** 恶搞重大自然灾害、意外事故、恐怖事件、战争等灾难场面，从而达到煽动情绪，引起对立的目的。

**4.2 传播不良价值观，如：**
- 宣扬流量至上，展示违背伦理道德的生活；
- 宣扬浪费、攀比、误导消费、拜金主义等不正确价值观；
- 实施或教唆"网络暴力"；
- 鼓励PUA撩妹、撩汉等相关话题；
- 煽动性别对立，性别歧视等。

**4.3 传播不文明行为：** 违反社会道德规范，展示、美化、宣扬不文明行为。

**4.4 展示传播低级趣味，低俗擦边等文字、图片和视频，例如：**
- 露出/聚焦敏感部位的图片以及敏感部位未打码的真人写真/艺术摄影/绘画作品等；
- 突出身材特征和敏感部位，含有性器官/性行为/性暗示的描述词汇，或强调男女交往的暧昧动作或描述的行为具有擦边倾向；
- av女优/男优、肉番等的文字介绍、图片；
- 描述人类性行为模式，绑缚与调教，施虐与受虐，支配与臣服的内容。

**4.5** 宣扬暴力、恶意谩骂、侮辱嘲讽他人等的内容。

**4.6** 其他有悖于社会公德与善良风俗的内容。

### 5、不良信息

我们希望让用户看到真实、科学的内容；不科学的内容可能会给人身体健康财产安全造成危害。不良信息包括但不限于：

**5.1 不规范医疗科普和行为，包括但不限于：**
- 科学性错误，如未经证实的养生、食膳、锻炼偏方秘方等；
- 片面/夸大描述药品、商品、产品、服务等医疗功效；
- 不符中医理论的内容，如宣传单味中药或药食同源食物治疗疾病；
- 无资质行医，指未取得合法执业许可资质的人员或机构，通过线上渠道提供诊疗服务，或引导用户至线下无资质诊所就医的行为。

**5.2 宣扬伪科学，包括但不限于：**
- 无科学依据的食物相克、违背自然规律等内容；
- 违背现有的大众认可的科学理论的内容；
- 宣扬封建迷信等内容。

**5.3 平台不倡导不友好的、可能会引起用户生理心理不适的内容，包括但不限于：**

5.3.1 观感不适内容，例如：
- 死亡事故现场图；
- 人体受到物理伤害的展示图，如烧伤、血流不止等；
- 器官、脏器、肌肉组织或者皮肤病、炎症、大面积溃烂等实例图，图片部分内容引人不适；
- 图片使用鬼怪正脸且较大等直观惊悚形象，引起读者不适；
- 动物撕咬进食或蛇虫等近距离特写、密集镜头，容易引人不适。

5.3.2 惊悚恐怖内容，例如：
- 恐怖灵异画面的图集、视频、音频内容；
- 真实涉及恐怖因素刑侦案件描述；
- 都市灵异事件讲述，民间悬疑故事；恐怖灵异悬疑的小说漫画游戏作品。

5.3.3 误导类标题，包括但不限于以下情况：
- 以浮夸的描述，煽动人群做/不做某行为的浮夸描述；
- 标题夸大事件严重程度、紧急程度、受影响面从而煽动情绪；
- 标题无中生有部分信息，以误导类的内容、描述，或者缺失、掩盖、歪曲事件具体信息和情况，使用户可能对时效性话题、事件产生错误理解或者不当关联的内容；
- 标题与内容完全不一致，误导用户；或标题与正文内容/中心思想无关，或关联内容篇幅较小。

5.3.4 关键信息缺失、不完整的标题，包括但不限于以下情况：
- 标题信息不完整，影响用户阅读；
- 只为吸引用户点击，标题故意隐藏关键信息，制造悬念或话说一半。

5.3.5 平台不倡导可能降低用户体验的内容，包括但不限于：
- 简单拼凑的无意义无逻辑内容，如重复文字堆砌的内容；
- 视频内容异常，如画质模糊、内容不完整、音画不相关等；
- 制作混剪、拼接、P图等可能误导或欺骗用户的内容；
- 封面图人物不完整、存在明显标记，或者画质不清等；
- 通过外链、公众号/服务号发送消息等方式，强制或诱导用户关注公众号/服务号，提供奖励（实物奖品、虚拟奖品等）。

**5.4 导流内容，包括但不限于以下情况：**
- 内容导流，即发布不完整内容，并通过二维码、文字、链接、图标等形式进行导流，内容的主要目的是导流用户至其他内容、小程序、其他公众号/服务号、视频号、第三方网页或应用；
- 互动导流，通过账号简介、内容、评论、公众号消息等各种途径，直接或间接嵌入个人联系方式、外部链接、二维码等引导信息，将用户引流至私人账号、外部页面或第三方平台；
- 多重嵌套导流，通过定向引流、多重跳转后，最终引导用户进行线上或线下交易，或发生可能骚扰用户、损害其交易安全与合法权益的行为。

### 6、营销推广

平台不鼓励仅以营销、广告为目的，缺乏实际价值的宣传和营销推广，甚至虚假宣传的内容；包括但不限于：
- 仅为组织宣传、商业推广广告的内容；
- 以促进销售为主要目的的广告内容，包含折扣优惠的内容；
- 强制或提供奖励诱导用户关注公众号/服务号。

### 7、低创作度内容

平台鼓励具有丰富信息含量的内容，欢迎为用户提供信息增量或情绪价值等价值的内容，不鼓励以各种形式进行疑似投机的低创作度内容，包括但不限于：

**7.1 同质化内容，频繁发布高度相似内容，缺乏信息增量，如：**
- 短期内，高频发布结构、框架、核心观点完全一致，或多要素高度重合的内容；
- 长期复用同一标题/封面图片，或对同一主题/素材反复加工翻炒，导致内容整体重复率过高。

**7.2 疑似搬运、抄袭等原创度较低的内容，如：**
- 疑似搬运内容，大量复述站内、外的内容，仅进行少量语序、标点等轻加工，且不含增量信息；
- 通过洗稿重组，保留他人原文核心逻辑/数据，仅重组语序/段落、替换近义词或增减无效信息；
- 机械拼接、组合多篇同主题内容，仅作简单拼凑、融合而未提供增量信息的内容。

**7.3 信息量不足，疑似无法为用户提供有效价值的内容，如：**
- 简单拼凑整合无意义信息，堆砌大量形容词、套话，缺乏实质性信息；
- 图文失配或看图说话内容，逻辑机械而混乱，形成图、文之间的信息冲突；
- 引用疑似不存在的数据、案例、言论作为依据，且未提供可验证来源。

**7.4 低价值AIGC内容，平台支持新技术辅助创作但不鼓励借用新技术直接生成、批量制造低信息量内容，如：**
- 通过AIGC等技术生成文章主体内容，占比显著高于人工创作部分，且并不声明AI辅助创作；
- 使用新技术，生成含事实错误、夸大表述或虚构数据的内容；
- 简单指令生成套路式/碎片式信息，从而进行大批量发布的低质灌水内容。

### 8、过时信息

过时信息指因时间推移或环境变迁，导致时效性、准确性或参考价值显著降低的内容，包括但不限于：
- 时效失效型内容，主要指过往实施或应用，但因修订废止而失去约束力的政策法规、行业标准等内容；
- 价值衰减型内容，对以往发生的社会事件或传言等内容进行二次传播，且没有新的进展或观点，不具备新的阅读价值，带来较差阅读体验的内容；
- 误导型内容，对历史事件通过语义扭曲、语境置换、暗示炒作等手段制造认知偏差，导致事件时间线模糊或误导为近期发生的行为。

### 9、虚假人设运营

指掩饰、虚构或盗用身份，导致账号对外身份特征与实名认证信息严重不符，可能使公众发生混淆、误解甚至受骗，包括但不限于以下行为：

**9.1 使用非本人信息，包括但不限于：**
- 冒用/盗用他人身份特征：盗用他人生活照、证件照、影像资料或个人隐私信息等身份特征，或未经授权利用AI换脸等新技术使用他人形象后，将其作为自身出镜形象或人设对外展示。

**9.2 制造虚假人设，包括但不限于：**
- 虚构身份形象：利用AI生成、修图等技术手段或其他方式，虚构完全不存在或与创作者自身严重不符的个人形象（包括外形、年龄、性别、职业背景或社会地位等），以达到误导公众、获取流量等不当目的；
- 仿冒特定身份：通过昵称、头像、简介、内容暗示等方式，假冒或恶意关联明星、公众人物、特定职业身份(如军人、专家)、特定社会关系(如朋友)，以获取不当关注或利益的。
